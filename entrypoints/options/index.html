🔒 Content Script Whitelist

The extension scans all web pages for magnet and torrent links. To improve performance and privacy, you can whitelist only specific domains where you need this feature. The extension will still work everywhere, but will only actively scan pages on whitelisted domains.

Example domains: piratebay.org, 1337x.to, torrentz.eu
