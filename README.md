# NAS Download helper

A browser extension project for Edge/Chrome that helps with downloading files from NAS devices.

## Files

- `background.js`
- `content.js`
- `manifest.json`
- `options.html`
- `options.js`
- `popup.html`
- `popup.js`
- `icons/`

## Getting Started

1. Open the `nas-download-helper` folder in VS Code.
2. Load the extension in Edge/Chrome via `edge://extensions` or `chrome://extensions`.
3. Enable Developer Mode and load the unpacked extension.
